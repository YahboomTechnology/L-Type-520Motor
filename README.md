# Introduction
The L-type 520 motor is a high-performance, compact DC motor designed for applications that require high torque and reliability. The reduction ratio is 1:40, with a high-precision Hall encoder, and all-metal gears inside. The speed after reduction is about 300RPM. Its unique L-shaped structure makes it easy to install in space-constrained environments while providing excellent power output.
